Usage

So far there are classes for Javascript and Ruby. They both work basically the same. View the source of this page to see the Javascript version in action.

Javascript

var options = {
  user: 'Username', // Github username
  repo: 'media',    // Repository name
  branch: 'master', // Branch (or SHA, etc.)
};

var media = new GithubMedia(options);
media.link_to('some_file.txt');
// http://github.com/Username/media/raw/master/some_file.txt

Ruby

options = {
  :user => 'Username',
  :repo => 'repo-name',
  :branch => 'master',
}

media = GithubMedia.new(options)
media.link_to('some_file.txt')
# => http://github.com/Username/media/raw/master/some_file.txt
